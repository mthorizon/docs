## Example API Call
Pulling top orders for a specific collection:

```bash class=wrap
curl --request POST \
--url https://marketplace-api.sequence.app/arbitrum-sepolia/rpc/Marketplace/GetTopOrders \
--header 'Content-Type: application/json' \
--data '{
    "collectionAddress": "0x1693ffc74edbb50d6138517fe5cd64fd1c917709",
    "currencyAddresses": [
        "0xa9c88358862211870db6f18bc9b3f6e4f8b3eae7"
    ],
    "orderbookContractAddress": "0xB537a160472183f2150d42EB1c3DD6684A55f74c",
    "tokenIDs": ["0","1", "2", "3", "4", "5"],
    "isListing": true,
    "priceSort": "DESC"
}'
```

For more resources on leveraging the Sequence Market Protocol, check out the [in-depth documentation](/api/marketplace/examples/orderbook-transactions).
