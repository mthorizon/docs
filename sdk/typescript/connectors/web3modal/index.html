Using Web3Modal

The Sequence wallet can easily be used with Web3Modal.

Try it live :)

A live Dapp example using Sequence + Wallet Connect with Web3Modal.

Source code is available too: https://github.com/0xsequence/demo-dapp-web3modal

Usage

The sequence wallet is compatible with wallet connect V2 and is therefore fully compatible with web3modal by default. Users can use the Scan button in the sequence wallet to connect.

Example

An example demo for the sequence Wallet + web3modal is available [here] (https://github.com/0xsequence/demo-dapp-web3modal)
